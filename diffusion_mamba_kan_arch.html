🎓 Training Phase (Learning to Denoise)

📥 Input Pair
Image: x (B, 1, H, W) - CT/MRI scan
Ground Truth: m₀ (B, C, H, W) - Clean segmentation mask
↓
🎲 Forward Diffusion Process DDPM
Gradually add Gaussian noise to ground truth mask over T timesteps
q(m_t | m_{t-1}) = 𝒩(m_t; √(1-βₜ)m_{t-1}, βₜI)

m_t = √(ᾱₜ)m₀ + √(1-ᾱₜ)ε,   ε ~ 𝒩(0, I)
Timesteps: T = 1000 (standard DDPM)
Noise Schedule: Linear or Cosine β_t
Output: Noisy mask m_t at random timestep t
↓
Learn θ

Encoder

Decoder

KAN Layers
Interpretable

Conditioning: Time embedding t + Image features x
Input: Concat[x, m_t, t_emb] → (B, C+3, H, W)
x: Original CT image (condition)
m_t: Noisy mask at timestep t
t_emb: Sinusoidal time embedding
Output: ε_θ(m_t, x, t) → Predicted noise
↓
📉 Training Loss
ℒ = 𝔼_{t,m₀,ε}[||ε - ε_θ(m_t, x, t)||²]

Simple MSE loss between true noise ε and predicted noise
Goal: Learn to predict noise ε at any timestep t
Advantage: Once trained, can denoise from pure noise → clean mask

🔮 Inference Phase (Generating Segmentation)

🔄 Reverse Diffusion (Denoising) DDIM Fast
Start from pure noise m_T ~ 𝒩(0,I), iteratively denoise to get clean mask m₀
m_{t-1} = √(ᾱ_{t-1}) pred_m₀ + √(1-ᾱ_{t-1}-σₜ²) ε_θ(m_t,x,t) + σₜε

where pred_m₀ = (m_t - √(1-ᾱₜ)ε_θ)/√ᾱₜ
Steps: T=50 (DDIM acceleration, 20x faster than DDPM)
Input: New CT image x (test)
Start: m_T ~ 𝒩(0,I) pure Gaussian noise
Iterate: t = T → T-1 → ... → 1
Output: Final clean segmentation mask m₀
↓
🎯 Final Segmentation Output
Output: m₀ (B, C, H, W) - Probabilistic segmentation
Post-process: Argmax over classes
Advantage: Can sample multiple times for uncertainty estimation
